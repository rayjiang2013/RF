What's New in Release 4.95

This section outlines the new features and changes from HLAPI 4.90-GA-Patch 1 to 4.95-GA.

Section Description Chapter Command Reference
HL API – BGP Added support for NGPF BGP Route Import (v4 and v6) and BGP Statistical Route Distribution. Chapter 3: BGP APIs emulation_bgp_route_config
HL API – Protocols Added support for the TLV editors available for the following NGPF protocols: DHCPv4 and v6 Client, Server and Relay Agent, ANCP. Chapter 34: NGPF APIs tlv_config
HL API – IPTV Added support for NGPF IPTV.

HL API – Connect Added HL API support for chassis chained using star topologies. Chapter 43: Session APIs connect
HL API – Connect A new -mode reconnect_ports was added to :ixia::connect command, which will reload port configuration (release all currently assigned ports then connect them back with the same vport assignment). Chapter 43: Session APIs connect
HL API – Interface Configuration Added support for configuring the ARP Refresh Timer for classic protocol interfaces. Chapter 43: Sessions APIs interface_config
HL API – Traffic Added granular HL API commands for traffic apply and regenerate actions. Chapter 46: Traffic APIs traffic_control
HL API – AppLibrary Added support for retrieving the TCP statistics for AppLibrary traffic. Chapter 46: Traffic APIs traffic_stats
HL API – Install Added support for running HL API scripts in 32bit and 64bit CentOS machines.
