Installing HLPython API on Windows System

The HLPython API package is provided in the same package as HLTAPI, which is in the form of an executable file of the following format:

HLTAPI_ <release> .exe

where <release> is the release designation.

To install HLPython API, do the following:

  1. Download the .exe file from the portal to your computer.
  2. Double-click this file to start the setup menu for HLTAPI installation. Follow the setup menu to complete the installation.

To access the HLTAPI library procs, use the following command:

package require Ixia

Please note that the HLTAPI product will install 32bit Python 2.7.6 in the following location :

<Ixia products installation folder>\Python\2.7.6

For example, C:\Program Files (x86)\Ixia\Python\2.7.6)

Similar to ActivePerl, the installer will create a Python folder in the Start menu. For example, C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Ixia\Python 2.7.x

If Python is already installed on the computer, the existing environment variables will not be changed during the installation. By default the previous settings are restored.
