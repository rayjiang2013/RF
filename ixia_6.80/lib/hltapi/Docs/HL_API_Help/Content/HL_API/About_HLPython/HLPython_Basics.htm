HLPython API Basics

This section is meant for readers who are new to HLPython API, but are familiarized with the HLTAPI product (if this is not the case, then refer to HLTAPI documentation present in the section, About This Guide). The purpose of this document is to provide a jump start for understanding the concept of High Level Python APIs, writing, running and debugging HLPython scripts.

This document only covers the basic command format, workflow and best practices. Once you are comfortable with the basic ideas of the APIs, reading the protocol command descriptions directly from the command manual and consulting the sample scripts provided in the HLT API installer, will help you in writing complex scenario scripts.
