HLTAPI Basics

This section is meant for readers who are either new to HLTAPI, or are versed in HLTAPI but new to Ixia products. The purpose of this document is to provide a jump start for understanding the concepts of APIs, getting help from the command manual and sample scripts in the distribution, and writing scripts.

This document only covers basic traffic generation and statistics retrieval. Once you are comfortable with the basic ideas of the API, reading protocol command descriptions directly from the command manual is straightforward.
