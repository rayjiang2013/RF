What's Deprecated

With the release of HLTAPI 4.70, the following features or capabilities were deprecated:

  • DCE-ISIS is no longer supported.

With the release of HLTAPI 3.50, the following features or capabilities were deprecated:

  • L2TPv3 APIs are no longer supported. (Chapter 15 removed from book and remaining chapters renumbered.)
  • IxAccess is no longer supported. API commands for L2TP and PPP are supported by using IxNetwork Tcl API instead of IxAccess Tcl API. IxAccess examples have been removed from Appendix A, Example APIs.

    Minor changes are required in the PPPoX and L2TP scripts to run on IxNetwork. See examples in Appendix A, IxNetwork L2TP and PPPoX scripts.
  • Deprecated options for specific commands using IxNetwork API are as follows:
    • pppox_config
      • enable_setup_throttling (ignored)
      • is_last_subport (no longer necessary; the needed configuration can be done without using this parameter)
    • pppox_stats-parameters that are no longer supported with IxNetwork:
      • mode session
      • csv_filename
    • l2tpox_config-parameters that are no longer supported with IxNetwork are as follows:
      • avp_framing_type
      • domain_group_map-the formatting of that list is different now (read the documentation for usage of this parameter)
      • ppp_server_step
      • vlan_user_priority_count
      • vlan_user_priority_step
      • cool_off_time
      • flap_repeat_count
      • flap_rate
      • hold_time
      • is_last_subport
      • l2tp_variant
      • qos_atm_clp
      • qos_atm_cr
      • qos_atm_efci
      • qos_byte
      • qos_fr_becn
      • qos_fr_cr
      • qos_fr_de
      • qos_fr_fecn
      • qos_ipv6_flow_label
      • qos_ipv6_traffic_class
      • qos_rate
      • qos_rate_mode
      • enable_setup_throttling
    • l2tpox_stats-parameters that are no longer supported with IxNetwork as as follows:
      • mode tunnel, session
      • csv_filename
    • traffic_config
      • traffic_generator ixaccess is no longer supported
