Namespaces_Packaging

A namespace is a collection of commands and variables. It encapsulates the commands and variables to ensure they do not interfere with the commands and variables of other namespaces.

Perl:

  • ::ixiahlt - This namespace is valid for HLP commands using IxNetwork legacy functionality
  • ::ixiahltgenerated - This namespace is valid for HLP commands using IxNetwork legacy functionality for FCoE, DCBX/LLDP, PTP, ESMC
  • ::ixiatcl – This namespace is used for calling Tcl native commands (set, list, array, etc)
  • ::ixiangpf – This namespace is valid for HLP commands using NGPF IxNetwork functionality

For details about the Protocols and their supported APIs please see Protocols and Required APIs.
