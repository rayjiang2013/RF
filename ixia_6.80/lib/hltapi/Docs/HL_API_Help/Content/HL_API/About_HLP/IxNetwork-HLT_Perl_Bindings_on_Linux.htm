IxNetwork/HLT Perl Bindings on Linux

The setup directions for Linux installation are as follows:

  1. Install the Ixia IxOS client. (Refer to IxOS user documentation)
  2. Install the Ixia IxNetwork tcl client. (Refer to IxNetwork user documentation)
  3. Install the Ixia hlt client software/tarball under $IXOSINSTALLDIR/lib/HLPAPI.
  4. Do the following steps as a root administrator:
  5. Install Active State Tcl8.4 on your linux computer. Active state's Perl Tcl integration module needs Active State Tcl.
  6. Install Active State Perl5.10 on your linux computer. Only the activestate perl is supported at this time for the Ix Network/HLT Perl.
  7. Install the Perl <-> Tcl package integration package for Perl through the active state ppm command:

# whereis ppm

ppm: /opt/ActivePerl-5.10/bin/ppm

# /opt/ActivePerl-5.10/bin/ppm install Tcl

From this point on, ensure that you are using /opt/ActivePerl-5.10/bin/perl. Otherwise, you are likely to get errors when loading tcl through the use Tcl; command in the Ixia Perl scripts and modules.
