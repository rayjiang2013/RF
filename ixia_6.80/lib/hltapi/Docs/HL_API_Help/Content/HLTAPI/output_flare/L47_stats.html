NAME - ::<namespace>:: L47_stats

DESCRIPTION

IxLoad

This command manages the statistics gathered during the test run. It can add multiple types of statistics, extract data, clear statistics from a previous run, or can add specific filters to the way the statistics are gathered. Per interface statistics can be collected with the use of the -ip_list parameter.

DESCRIPTION

Specifies how statistics for multiple ports are combined. Dependencies: mode are as follows:

Valid options are:

addsum

add all of the statistics together.

max

determined the maximum value.

min

determine the minimum value.

average

determine the average value.

rate

determine the rate of change of the sum of all the statistics.

maxrate

determine the maximum rate.

minrate

determine the minimum rate.

determine the average rate.

This type is for use with weighted statistics. The statistics descriptions indicate whether they are weighted or not.
