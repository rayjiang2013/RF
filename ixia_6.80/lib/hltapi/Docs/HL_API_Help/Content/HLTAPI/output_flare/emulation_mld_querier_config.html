-mode

Supported platforms Details

IxNetwork-NGPF [M]

DESCRIPTION

This option defines the action to be taken.

Valid options are:

create

Creates the MLD querier router on the Ixia interface. Requires the use of -handle.

modify

Modifies the MLD querier router on the Ixia interface. Requires the use of -handle.

delete

Deletes the MLD querier router on the Ixia interface. Requires the use of -handle.

disable

The MLD querier router will be disabled. Requires the use of -handle.

enable

Enables the MLD querier router on the Ixia interface. Requires the use of -handle.

DEFAULT

 None

-discard_learnt_info

Supported platforms Details

IxNetwork-NGPF

DESCRIPTION

When -discard_learnt_info is 0, the emulated Querier maintains a complete record state for received reports and send queries (based on timer expiry for received groups and sources). If -discard_learnt_info is 1, the Querier does not maintain any database and only sends periodic general queries. The specific query group/source record information is not calculated based on any earlier received report, but is based only on the last received report.

DEFAULT

1

Valid in combination with parameter(s)
