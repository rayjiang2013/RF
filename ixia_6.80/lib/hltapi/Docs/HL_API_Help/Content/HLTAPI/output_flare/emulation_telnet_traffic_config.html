NAME - ::<namespace>:: emulation_telnet_traffic_config

DESCRIPTION

IxLoad

The command is used to construct the model for client or server traffic to be applied, or respectively, handled during the test. This traffic is composed by commands created by client TELNET agents which are handled by server TELNET agents. Multiple agents can be added to form a TELNET traffic.

[M] indicates if the argument is mandatory on the supported platform.

-handle

Supported platforms Details

IxLoad

DESCRIPTION

Handle of a telnet configuration when -property traffic and -mode add. Or handle of a traffic property , used when -property is agent and -mode is add. Also the handle of a traffic, agent or command property when -property is respectively traffic, agent and -mode is modify | remove | enable | disable.Dependencies: -property agent; -property traffic command with -mode modify.

DEFAULT

 None

-target

Supported platforms Details

IxLoad

DESCRIPTION

Specifies where this traffic will be added: on the client side or on the server side. Values: client or server.Dependencies: -property traffic with -mode add.

DEFAULT

 None

-property

Supported platforms Details

IxLoad [M]

DESCRIPTION

Defines types of properties of the TELNET traffic configuration. Command handles operations required for defining commands for the telnet agents.

Valid options are:

traffic

handles operations required for defining the telnet

agent

handles operations required for defining the emulated

 None

-command

Supported platforms Details

IxLoad

DESCRIPTION

The command sent to the telnet server.Dependencies: property command with mode add | modify. open - Open a connection to a Telnet server.login - Login to the Telnet server, sending the user name.password - Send a password.send - Send an arbitrary string.exit - Close the session with the Telnet server.think - Wait for a random period of time within a specified range.

DEFAULT

 None

-expect

Supported platforms Details

IxLoad

DESCRIPTION

This option is dependent of the -command option value.Dependencies: property command with mode add | modify.If -command is:open - The expected response from the command. (DEFAULT = "login:")login - The expected response from the command. (DEFAULT = "Password::")password - The expected response from the command. If no value is specified then the default_command_prompt of the client agent is used.send - The expected response from the command. If no value is specified then the default_command_prompt of the client agent is used.think - The upper limit of a randomly chosen sleep, expressed in

 None

Supported platforms Details

(DEFAULT = "")login - The login name to send to the Telnet server. password - The password to send to the Telnet server. send - The string to send to the Telnet server. exit - The string to send to the Telnet server to exit the Telnet think - The lower limit of a randomly chosen sleep, expressed in Dependencies: property command with mode add | modify.

DEFAULT

 None
