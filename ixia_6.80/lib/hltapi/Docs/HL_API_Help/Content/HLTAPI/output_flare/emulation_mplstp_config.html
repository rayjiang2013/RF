NAME - ::<namespace>:: emulation_mplstp_config

DESCRIPTION

IxNetwork, IxOS/IxNetwork-FT

This command is used to create/modify/delete/enable/disable a MPLS-TP router or router interface. Each MPLS-TP router can have multiple MPLS-TP router interfaces. Each router interface is associated with a protocol interface.

ARGUMENTS

[M] indicates if the argument is mandatory on the supported platform.

-mode

Supported platforms Details

IxNetwork

DESCRIPTION

Action to perform. This parameter is supported using the following APIs: IxN. create - Create new MPLS-TP routers or router interfaces or both. Param -port_handle is needed for creating routers (with possible multiple interfaces, see -interface_count). Param -router_handle is needed for creating interfaces on a specified router. modify - Modify MPLS-TP router or router interface specified with -router_handle or -interface_handle parameters. enable - Enable MPLS-TP router or router interface specified with -router_handle or -interface_handle parameters. disable - Disable MPLS-TP router or router interface specified with -router_handle or -interface_handle parameters. delete - Remove MPLS-TP router or router interface specified with -router_handle or -interface_handle parameters.

Supported platforms Details

DESCRIPTION

The step for Router ID. If this is specified, -router_id cannot be a list. APIs: IxN.

DEFAULT

0.0.1.0

RETURN VALUES

Supported Platform Details

IxNetwork

status

$::SUCCESS | $::FAILURE

log

On status of failure, gives detailed information.

router_handles

On mode create, list of router handles. On modes modify/enable/disable the router affected.

interface_handles

On mode create, list of router interface handles for a specific <router_handle> On modes modify/enable/disable the interface affected.
