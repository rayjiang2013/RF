NAME - ::<namespace>:: emulation_cfm_vlan_config

DESCRIPTION

IxNetwork, IxOS/IxNetwork-FT

This procedure configures Vlan properties on the bridge specified with bridge_handle. The mac range handles returned with 'mac_range_handles.$vlan' key can be used as endpoints for traffic. Traffic is configured using ::::traffic config using '-circuit_endpoint_type ethernet_vlan' and '-circuit_type none'. This protocol is available only when using IxTclNetwork. The GUI equivalent is in the section Protocols/Carrier Ethernet where CFM/Y.1731/PBB-TE is enabled.
