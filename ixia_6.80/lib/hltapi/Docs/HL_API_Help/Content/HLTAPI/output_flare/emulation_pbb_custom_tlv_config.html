-bridge_handle

Supported platforms Details

IxNetwork [M]

DESCRIPTION

The value of the -bridge_handle parameter must be a bridge handle returned by ::::emulation_pbb_config procedure. This parameter is supported using the following APIs: IxN.

DEFAULT

 None

-mode

Supported platforms Details

IxNetwork [M]

DESCRIPTION

Action to perform. This parameter is supported using the following APIs. Valid choices are:

Valid options are:

create

Create new Custom TLV on -port_handle.

modify

Modify Custom TLV specified with -handle parameter.

enable

Enable Custom TLV specified with -handle parameter.

disable

Disable Custom TLV specified with -handle parameter.

remove

Remove Custom TLV specified with -handle parameter.

DEFAULT

 None

Supported platforms Details

IxNetwork

DESCRIPTION

DEFAULT
