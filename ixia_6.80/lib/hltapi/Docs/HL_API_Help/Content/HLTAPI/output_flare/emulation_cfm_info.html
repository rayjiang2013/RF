-mode

Supported platforms Details

IxNetwork [M]

DESCRIPTION

Deterines whether statistic filters are set or if statistic are gathered and returned in keyed lists. The supported parameters are:

Valid options are:

set_filters

Configure statistic filters for bridge specified with -bridge_handle parameter.

get_ccm

Return CMM learned information keys for bridge specified with -bridge_handle parameter.

get_cfm_itu

Return CFM ITU delay measurement informaton keys for the CFM bridge specified with -bridge_handle parameter.

get_cfm_lt

Return CFM Link Trace information keys for bridge specified with -bridge_handle parameter.

get_cfm_lb

Return CFM Loopback information keys for bridge specified with -bridge_handle parameter.

get_periodic_oam_lt

Return OAM link trace learned information keys for bridge specified with -bridge_handle parameter.

get_periodic_oam_lb

Return OAM loopback learned information keys for bridge specified with -bridge_handle parameter.

get_periodic_oam_dm

Return OAM delay measurement learned information keys for bridge specified with -bridge_handle parameter.

get_all

Return all keys available.

DEFAULT

 None

Supported platforms Details

IxNetwork

DESCRIPTION

CFM Bridge handle. Statistic filters will be set on this bridge handle if -mode is set_filters. Statistic keys will be returned from this bridge handle for all other modes.This parameter is supported using the following APIs: IxN.
