-mode

Supported platforms Details

IxNetwork [M] , IxOS/IxNetwork-FT [M]

DESCRIPTION

This option defines the action to be taken.

Valid options are:

start

Starts the MLD host server, requires the use of -port_handle.

stop

Stops the MLD host server, requires the use of -port_handle.

restart

Restarts the MLD host server, requires the use of -port_handle.

join

This action works with -handle or -port_handle parameter. It will enable all the group ranges on the specified port/mld host.

leave

This action works with -handle or -port_handle parameter. It will disable all the group ranges on the specified port/mld host.

DEFAULT

 None

IxNetwork-NGPF [M]

DESCRIPTION

This option defines the action to be taken.

Valid options are:

start

Starts the MLD host, requires the use of -handle.

stop

Stops the MLD host, requires the use of -handle.

restart

Restarts the MLD host, requires the use of -handle.

join

If -group_member_handle is provided join specified group pool(s). If -handle is provided, enable all groups on this session. If -port_handle is provided, enable all groups on all sessions for that port.

leave

If -group_member_handle is provided leave specified group pool(s). If -handle is provided, disable all groups on this session. If -port_handle is provided, disable all groups on all sessions for that port.

rejoin

Sends another Join message to the Querier.

releave

Sends another Leave message to the Querier.

mld_send_specific_query

MLD Send Specific Query

mld_resume_periodic_gen_query

MLD Resume Periodic Gen Query

mld_stop_periodic_gen_query

MLD Stop Periodic Gen Query

DEFAULT

 None

-handle

Supported platforms Details

IxNetwork, IxOS/IxNetwork-FT

DESCRIPTION

If -mode is join or leave, all group pools belonging to the MLD session specified with this option will join or leave.

DEFAULT

 None

IxNetwork-NGPF

DESCRIPTION

If -mode is join or leave, all group pools belonging to the MLD session specified with this option will join or leave. Example :- ::ixiangpf::emulation_mld_control -handle $device1handle -mode [start or stop]

DEFAULT

 None

-port_handle

Supported platforms Details

IxNetwork, IxOS/IxNetwork-FT

DESCRIPTION

This option is required to specify the port where to take the action.

DEFAULT

 None

DESCRIPTION

DEFAULT

 None

-group_member_handle

Supported platforms Details

IxNetwork, IxOS/IxNetwork-FT

DESCRIPTION

If -mode is join or leave this option specifies the group pools to join or leave.

DEFAULT

 None

IxNetwork-NGPF

DESCRIPTION

If -mode is join or leave this option specifies the group pools to join or leave.

DEFAULT

 None

-source_member_handle

Supported platforms Details

IxNetwork-NGPF

DESCRIPTION

If -mode is join or leave this option specifies the source pools to join or leave. If -port_handle is provided along with -source_member_handle and -mode join/leave, the specified source will be enabled/disabled. If -source_member_handle is provided only with -mode join/leave, the source will execute join/leave.

DEFAULT

 None

-start_group_address

Supported platforms Details

IxNetwork-NGPF

DESCRIPTION

Start group address for the mld host on which the action defined by the mode parameter will be applied

DEFAULT

ff03:0:0:0:0:0:0:1

-group_count

Supported platforms Details

IxNetwork-NGPF

DESCRIPTION

Group count for the mld host on which the action defined by the mode parameter will be applied

DEFAULT

-start_source_address

Supported platforms Details

IxNetwork-NGPF

DESCRIPTION

Start source address for the mld host address on which the action defined by the mode parameter will be applied

DEFAULT

-source_count
