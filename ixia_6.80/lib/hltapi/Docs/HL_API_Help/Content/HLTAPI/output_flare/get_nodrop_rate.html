NAME - ::<namespace>:: get_nodrop_rate

DESCRIPTION

IxOS/IxNetwork-FT

This is a generic nodrop rate calculation routine. Traffic streams are setup before calling this proc. The basic function of the proc is to determine max rates at which the streams arereceived without any loss.

ARGUMENTS

[M] indicates if the argument is mandatory on the supported platform.

-stream_id

Supported platforms Details

IxOS/IxNetwork-FT [M]

DESCRIPTION

The list of streams that nodrop rate calculation is peformed

DEFAULT

 None

-max_rate

Supported platforms Details

IxOS/IxNetwork-FT [M]

DESCRIPTION

Either a number or a list of max stream rates. If this is a list, it must be aligned with the stream list. If this is a number, the max rates are assumed to be the same.

DEFAULT

 None

-tx_port_handle

Supported platforms Details

IxOS/IxNetwork-FT [M]

DESCRIPTION

The list of TX port handles for the streams. Must be aligned with the stream list.

DEFAULT

 None

-rx_port_handle

Supported platforms Details

IxOS/IxNetwork-FT [M]

DESCRIPTION

The set of port handles that are the receiving sides of the streams. This list does not need to be aligned with the stream list.

DEFAULT

 None

-stream_mode

Supported platforms Details

IxOS/IxNetwork-FT

DESCRIPTION

The operation mode for the calculation. Even mode will scale all the streams at the same rate, while uneven mode will scale the streams individually.

DEFAULT

even

-min_percent

Supported platforms Details

IxOS/IxNetwork-FT

DESCRIPTION

The lower bound of the nodrop rate calculation, in percentage. Default is 0.

DEFAULT

0

-run_time_sec

Supported platforms Details

IxOS/IxNetwork-FT

DESCRIPTION

Traffic run time for each iteration. Default is 10 (seconds).

DEFAULT

10

-tolerance

Supported platforms Details

IxOS/IxNetwork-FT

DESCRIPTION

Granularity of the calculation, in percentage.Default is 0.5 (0.5%).

DEFAULT

0.5

-poll_timeout_sec

Supported platforms Details

IxOS/IxNetwork-FT

DESCRIPTION

Timeout of polling for zero RX rate. Default is 100 (seconds).

DEFAULT

100

RETURN VALUES

Supported Platform Details

IxOS/IxNetwork-FT

status

$::SUCCESS | $::FAILURE

log

If status is failure, contains more information

nodrop_percent

Nodrop rate in percentage for even mode

<stream_id>.nodrop_percent

Nodrop rate in percentage for uneven mode, for that stream

<stream_id>.nodrop_pps

Nodrop rate in packet per second
