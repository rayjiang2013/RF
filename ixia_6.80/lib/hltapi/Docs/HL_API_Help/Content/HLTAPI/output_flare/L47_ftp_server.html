Supported platforms Details

IxLoad

DESCRIPTION

The port number to which the FTP server will respond. To specify multiple listening ports, enter the port numbers, separated by commas (,). You can specify up to 50 listening ports. (Default = 21).Dependencies: -property server | agent; mode add | modify

DEFAULT

21)

Supported platforms Details

IxLoad

DESCRIPTION

Handle of a ftp property.server_handle - When an agent is created -property agent; -mode add. - When a server is removed -property server; -mode removeagent_handle - When an agent is modified -property agent; -mode modify. - When an agent is enabled -property agent; -mode enable. - When an agent is disabled -property agent; -mode disable. - When an agent is removed -property agent; -mode remove. - When a real_file is added -property real_file; -mode addreal_file_handle - When a real_file is modified or removed: -property real_file; -mode modify | remove

DEFAULT

-property

Supported platforms Details

IxLoad

DESCRIPTION

Defines the object that will be configured.server = a server object and an agent object attached to the server object are createdagent = an agent object is created and it is associated with the server object specified by -handle "a server object". The server object can also be one created by another protocol.

DEFAULT

 None

-tos

Supported platforms Details

IxLoad

DESCRIPTION

If tos_enable is true, this option specifies the IP Precedence / TOS (Type of Service) bit setting and Assured Forwarding classes. (Default="Best Effort 0x0"). If you want to specify the standard choices that are in the GUI, you can use a string representation. To specify any of the other 255 TOS values, specify the decimal value.Dependencies: -property server | agent; mode add | modify; tos_enable 1Best Effort (0x0) - (Default) routine priorityClass 1 (0x20) - Priority service, Assured Forwarding class 1Class 2 (0x40) - Immediate service, Assured Forwarding class 2Class 3 (0x60) - Flash, Assured Forwarding class 3Class 4 (0x80) - Flash-override, Assured Forwarding class 4Express Forwarding (0xA0) - Critical-ecpControl (0xC0) - Internet-control

DEFAULT

0)

-tos_enable

Supported platforms Details

IxLoad

DESCRIPTION

Enables the setting of the TOS (Type of Service) bits in the header of the FTP packets. Use the tos option to specify the TOS bit setting.0=(default) TOS bits not enabled.1=TOS bits enabled.Dependencies: -property server | agent; mode add | modify

DEFAULT

0)

-rf_name

Supported platforms Details

IxLoad

DESCRIPTION

Any linux file name (e.g. /client_file1 or /dnd_files/new_file)Dependency: -property real_file

DEFAULT

 None
