ARGUMENTS

[M] indicates if the argument is mandatory on the supported platform.

-control_range_handle

Supported platforms Details

IxNetwork

DESCRIPTION

For -mode create or modify, parameter -control_range_handle handle specifies the TWAMP control range to which the TWAMP test range is related.

DEFAULT

 None

-handle

Supported platforms Details

IxNetwork

DESCRIPTION

For -mode create, parameter -handle specifies the IP endpoint range (previously configured with interface_config) where the TWAMP test range is to be configured. For -mode modify/enable/disable/remove, this is the TWAMP test range handle to be modified/enabled/disabled/removed.

DEFAULT

 None

-mode

Supported platforms Details

IxNetwork

DESCRIPTION

Action to perform.

DEFAULT

create

-num_pkts

Supported platforms Details

IxNetwork

DESCRIPTION

Number of packets to be sent by the Session-Sender

DEFAULT

1000

-padding_with_zero

Supported platforms Details

IxNetwork

DESCRIPTION

Per RFC465, data in the packets is random, unless it is configured to be zero.

DEFAULT

0

-pkt_length

Supported platforms Details

IxNetwork

DESCRIPTION

Packet length including padding length as defined by the RFC4656.

DEFAULT

64

-pps

Supported platforms Details

IxNetwork

DESCRIPTION

Rate at which packets will be sent

DEFAULT

10

-session_reflector_port

Supported platforms Details

IxNetwork

DESCRIPTION

Port on which the reflector receives the packets from the stream initiated by Session-Sender

DEFAULT

12001

-session_reflector_port_step

Supported platforms Details

IxNetwork

DESCRIPTION

The port step used by the reflector port within this range.

DEFAULT

0

-session_sender_port

Supported platforms Details

IxNetwork

DESCRIPTION

Source Port of the stream initiated by Session-Sender

DEFAULT

11001

-session_sender_port_step

Supported platforms Details

IxNetwork

DESCRIPTION

The port step used by the session sender port.

DEFAULT

1

-test_sessions_count

Supported platforms Details

IxNetwork

DESCRIPTION

Number of test sessions.

DEFAULT

1

-timeout

Supported platforms Details

IxNetwork

DESCRIPTION

DEFAULT

1
