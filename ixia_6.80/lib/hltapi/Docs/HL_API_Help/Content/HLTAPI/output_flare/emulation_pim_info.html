NAME - ::<namespace>:: emulation_pim_info

DESCRIPTION

IxNetwork, IxOS/IxNetwork-FT

This command is used to retrieve aggregate statistics about PIM from a port, and learned CRP_BSR info.

IxNetwork-NGPF

This command is used to retrieve aggregate statistics about PIM from a port, and learned CRP_BSR info.

SYNOPSIS - ::<namespace>::emulation_pim_info

List arguments in the alphabetical order or in the reverse alphabetical order.

CHOICES aggregate stats_per_device_group stats_per_session learned_crp clear_stats

-

]

ARGUMENTS

[M] indicates if the argument is mandatory on the supported platform.

-handle

Supported platforms Details

IxNetwork-NGPF [M]

DESCRIPTION

For -mode aggregate, this parameter should be provided with an emulated router or join/prune or source handle, but the command will return per port aggregated stats. For -mode learned_crp, this parameters should be provided with an emulated router or interface (for the emulated router) and the command will return per interface learned CRP and BSR info.

DEFAULT

 None

-mode

Supported platforms Details

IxNetwork-NGPF

DESCRIPTION

Using this argument, you can return aggregate statistics or learned crp-bsr info

Valid options are:

aggregate

stats_per_device_group

stats_per_session

learned_crp

clear_stats

DEFAULT

aggregate
