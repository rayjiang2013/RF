About This Guide

This guide describes the Ixia implementation of the High Level Python Application Programming Interface (HLPython API). The intended audience consists of test technicians who configure Ixia test equipment and software.

The intent of the HLPython API is to make the use of the test equipment easier to automate. The HLPython API hides Ixia’s lower level API, greatly reducing test script changes when Ixia’s lower level API changes in newer releases. Any differences are hidden under the HLPython API, making them completely transparent to the user, and thus requiring no changes to the test scripts.
