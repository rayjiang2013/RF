Installing HLPython API on Linux Systems

The following section describes the process of installing HLPython API on a Linux and Unix systems.

HLPython API is included in the installer of the HLTAPI product, so you need to install HLTAPI in order to access HLPython API libraries.

In order to install HLTAPI:

  1. Download either the ZIP,BIN or TAR file from the portal.
  2. Extract and copy all folders and files from the ZIP or TAR file into the lib folder of the IxOS client installation. For example, if the IxOS client is installed in /home/ixia/ixos6.70, the HLTAPI should be extracted to /home/ixia/ixos6.70/lib.

    3. Alternatively, create an HLTAPI only folder and extract the files there. In this case, set the TCLLIBPATH to point to both IXIA client installation and the HLTAPI installation.

  3. Add the following environment variables to point to the HLTAPI package:
    • IXIA_HOME: where the IxOS client is installed.
    • IXIA_VERSION–HLTSET value: This is one of several preset options. For more information on how to set this variable, see Setting the IXIA_VERSION Variable.
    • TCLLIBPATH: path to the IxTclHal and HLTAPI library where the package can be found.

See the following example:

IXIA_HOME=/home/ixia/ixos6.70

IXIA_VERSION=HLTSET166

TCLLIBPATH=$IXIA_HOME/lib

If HLTAPI is installed in its own folder, set TCLLIBPATH to also point to the HLTAPI folder. For example:

HLTAPI_HOME=/home/HLTAPI4.90

TCLLIBPATH="$IXIA_HOME/lib $HLTAPI _HOME"

To access HLTAPI library procedures, issue the following commands in Tcl or expect shell:

package require Ixia

LD_LIBRARY_PATH: folders to search for shared libraries.

LD_LIBRARY_PATH=$IXIA_HOME/lib:$TCL_HOME/lib:$LD_LIBRARY_PATH

PATH=$IXIA_HOME/bin:$TCL_HOME/bin:$PATH

 

For HLT Python API,

The Python version installed by HLT can be accessed using the ixiapython command, which is a symbolic link created while installing HLT. In case you have installed multiple versions of Python on your system, the ixiapython command runs the Python version added with the latest Ixia application installed.

To run your script using the latest version of Python installed use the following syntax:

ixiapython <script name>

To run your script using a different version of Python, use the ixiapython command from the specific Python installation folder. You can use the following syntax:

~/Ixia/<hltapi version 1>/python/bin/ixiapython <script name>

HLPython API requires the following Tcl lib packages:

  • Tclx: this is used for keyed lists. The HLT commands need this to return a keyed list. This library is usually included in third-party Tclsh like Active Tcl 8.4.
  • Tcl8.5
  • Tk8.5
  • libXft
  • libXScrnSaver

Sample packages installation: “yum install libXft libXScrnSaver tclx tcl tk “

Please note that LD_LIBRARY_PATH needs to be updated to include the paths to the libraries.
