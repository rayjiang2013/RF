Installing Your Own Copy

Install HLTAPI in your own computer under Windows or Linux OS, or a home folder under Solaris.

To install the library on a computer, simply double-click the HLTAPI_<release>.exe file, which starts the setup menu for HLTAPI installation.

For Linux users, create a folder under $IXIA/lib, where $IXIA is the folder name for Ixia software installation. Next, extract and copy all folders and files from the ZIP or TAR file to $IXIA/lib.

TclX extension is required. If this is not yet installed (open the wish or Tcl shell, type package require Tclx, and check for error), the TclX extension can be downloaded from the Internet. Create a new folder for TclX just like HLTAPI in the location mentioned previously.

For ATS users, or those who do not have root access to install HLTAPI at the preferred location, pick any location (for example, /users/sjlin/IxiaSoftware/hltapi2.60) for HLT files. Next add the path to the TCLLIBPATH environment. For example, in .bashrc:

export TCLLIBPATH="$TCLLIBPATH /users/sjlin/IxiaSoftware/hltapi2.60"

or in .cshrc:

setenv TCLLIBPATH "$TCLLIBPATH /users/sjlin/IxiaSoftware/hltapi2.60"

Refer to Installing HLTAPI section for more information. Also refer to A Script Example.
