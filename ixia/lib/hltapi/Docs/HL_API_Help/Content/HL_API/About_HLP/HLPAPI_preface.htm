About This Guide

This guide describes the Ixia implementation of the High Level Perl Application Programming Interface (HLPAPI). The intended audience consists of test technicians who configure Ixia test equipment and software.

The intent of the HLPAPI is to make the use of the test equipment easier to automate. The HLPAPI hides Ixia’s lower level API, greatly reducing test script changes when Ixia’s lower level API changes in newer releases. Any differences are hidden under the HLPAPI, making them completely transparent to you, and thus requiring no changes to the test scripts themselves.

The Ixia HLPAPI adheres to the following requirements:

  • It is a self-contained package containing everything needed to use the library. The individual test technician uses the Perl ‘use’ command to use the Ixia HLPAPI calls.
  • It is distributed from a link off the ATS Web site.
