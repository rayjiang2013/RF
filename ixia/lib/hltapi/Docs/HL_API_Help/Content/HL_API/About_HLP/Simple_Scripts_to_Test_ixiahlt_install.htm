Simple Scripts to Test ixiahlt install

Here is a simple script example to test ixiahlt installation.

use lib '/home/mhoegeman/ixos/lib/library/common/ixia_hl_lib-5.50';

use ixiatcl;

print("ixiatcl in place\n");

$ixiatcl::interp->Eval('lappend ::auto_path /home/mhoegeman/ixos/lib');

print("IxTclNetwork:" . $ixiatcl::interp->Eval('package require IxTclNetwork') .

"\n");

print("Done\n");

#$ixiatcl::interp->Eval('exit 0');

use lib '/home/mhoegeman/ixos/lib/library/common/ixia_hl_lib-5.50';

use ixiahlt;

print ( $ixiatcl::interp->Eval("info commands ixiahlt::fcoe*") . "\n");

#$ixiatcl::interp->Eval("exit 0");

Perl code is generated by making a very thin Perl binding to the generated tcl commands through the use of the the Perl Tcl package. We recommend you to use the active state's active-perl suite 5.10. Perl and tcl environments fulfills the following requirements:

  • Perl 5.10

Ensure that you use the 32-bit version of Perl even on 64-bit computers. The ixnetwork client dll does not load if you are using the 64-bit version of Perl.

  • Using the Tcl Module Version: 0.97 or later
  • Using a Tcl interpreter that is 8.4.7 or later

Active-perl comes bundled with the Perl Tcl package.
