Notes on Connect parameters that High-Level Scripts Use

  • ixnetwork_tcl_server - This refers to the IxNetwork application started in Tcl Server mode. Example: app_server_name:8009. 8009 is the Tcl port that IxNetwork uses. If the port used is 8009, it can be omitted as this is the default port.

  • device - This refers to the IxOS chassis being used by the IxNetwork configuration and can be omitted when using offline ports or when doing a session resume, in which case the current chassis from IxNetwork will not be changed.

  • tcl_server - This refers to the IP of the IxOS Tcl Server application.
