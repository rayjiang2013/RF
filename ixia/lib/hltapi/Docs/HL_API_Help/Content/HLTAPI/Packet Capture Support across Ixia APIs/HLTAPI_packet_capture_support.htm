Packet Capture Support across Ixia APIs

Starting HLT 4.40, support for IxNetwork Analyzer packet capture has been added.

This support was added for the following pre-existing commands in order to maintain backwards compatibility:

  • ::ixia::packet_config_buffers
  • ::ixia::packet_config_filter
  • ::ixia::packet_config_triggers
  • ::ixia::packet_control
  • ::ixia::packet_stats

The table shown in the Support for Packet Capture section below, explains the support for packet capture across Ixia APIs for the pre-existing commands.

New commands have been added for user defined stats ixia::uds_config and ixia::uds_filter_pallette_config that include all IxNetwork Analyzer UDS functionality and are also supported by HLT Scriptgen.
