DESCRIPTION

IxNetwork

This procedure retrieves information about the ELMI protocol.

ARGUMENTS

[M] indicates if the argument is mandatory on the supported platform.

-mode

Supported platforms Details

IxNetwork

DESCRIPTION

Action to be performed on the ports or the handles supplied. Following choices are supported.

Valid options are:

aggregate

returns per port aggregated stats

evc_status_learned_info

returns evc status learned info

uni_status_learned_info

returns uni status learned info

lmi_status_learned_info

returns lmi status learned info

uni_learned_info

returns all status learned info

DEFAULT

aggregate

-port_handle

Supported platforms Details

IxNetwork

DESCRIPTION

The port handle of the port on which the action specified by the -mode argument has to be performed.

DEFAULT

 None

-handle

Supported platforms Details

IxNetwork

DESCRIPTION

The ELMI handle on which the action specified by the -mode argument has to be performed.

DEFAULT

 None
