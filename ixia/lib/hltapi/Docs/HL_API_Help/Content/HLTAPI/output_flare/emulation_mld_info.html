-mode

Supported platforms Details

IxNetwork-NGPF [M]

DESCRIPTION

The statistics that should be retrieved for MLD host and querier.

Valid options are:

stats

Gather MLD aggregated statistics.

aggregate

Gather MLD aggregated statistics.

stats_per_device_group

Gather MLD statistics per Device Group.

stats_per_session

Gather MLD statistics per Session.

clear_stats

Clear MLD statistics.

learned_info

Get Learned Info

DEFAULT

 None

-port_handle

Supported platforms Details

IxNetwork-NGPF

DESCRIPTION

This option is required to specify the port where to take the action.

DEFAULT

 None

-timeout

Supported platforms Details

-handle

Supported platforms Details

IxNetwork-NGPF

DESCRIPTION

MLD session handle for which statistics are gathered. The handle is an emulated MLD router object reference. Example :

  • ::ixiangpf::emulation_mld_info -mode stats -handle $devicehandle -type [both or host or querier]
  • ::ixiangpf::emulation_mld_info -mode learned_info -handle $mldQuerierHandle

DEFAULT

 None

-type

Supported platforms Details

IxNetwork-NGPF

DESCRIPTION

The type of aggregated statistics to be gathered. This option is available only when using the IxNetwork (IxTclNetwork package) 6.30 or later. The option is NOT supported by IxNetwork-FT (IxTclProtocol package). Valid only for -mode stats.

Valid options are:

host

statistics gathered from IxNetwork for the MLD sessions configured using emulation_mld_config.

querier

statistics gathered from IxNetwork for the MLD sessions configured using emulation_mld_querier_config.

both

both host and querier agregated statistics will be retrieved.

DEFAULT

host
