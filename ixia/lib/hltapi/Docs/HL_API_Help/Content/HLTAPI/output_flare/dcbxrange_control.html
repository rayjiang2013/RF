-handle

Supported platforms Details

IxNetwork [M]

DESCRIPTION

 

A handle returned via a ::<namespace>::dcbxrange_config command. This is the object on which the action will take place.

DEFAULT

 None

-action

Supported platforms Details

IxNetwork [M]

DESCRIPTION

 

Action to be executed.

Valid options are:

abort

abort all sessions on the same port with the target endpoint designated by -handle. The control is returned when the operation is completed.

abort_async

abort all sessions on the same port with the target endpoint designated by -handle. The control is returned immediately. The operation is executed in the background.

start

start and/or negotiate session for the target endpoint designated by -handle.

connect

start and/or negotiate session for the target endpoint designated by -handle.

stop

stop and/or release session for the target endpoint designated by -handle.

disconnect

stop and/or release session for the target endpoint designated by -handle.

DEFAULT

 None
