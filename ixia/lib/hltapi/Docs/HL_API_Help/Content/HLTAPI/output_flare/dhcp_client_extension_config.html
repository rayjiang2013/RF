-handle

Supported platforms Details

IxNetwork [M] , IxOS/IxNetwork-FT [M]

DESCRIPTION

The protocol stack on top of which DHCpv6 client extension is to be created (for -mode add).The DHCpv6 client extension that needs to be modified/removed/enabled/disabled (for -mode modify/remove/enable/disable).

DEFAULT

 None

IxNetwork-NGPF [M]

DESCRIPTION

The protocol stack on top of which DHCpv6 client extension is to be created (for -mode add). The DHCpv6 client extension that needs to be modified/removed/enabled/disabled (for -mode modify/remove/enable/disable). When -handle is provided with the /globals value the arguments that configure global protocol setting accept both multivalue handles and simple values. When -handle is provided with a a protocol stack handle or a protocol session handle, the arguments that configure global settings will only accept simple values. In this situation, these arguments will configure only the settings of the parent device group or the ports associated with the parent topology.

 None
