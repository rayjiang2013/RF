NAME - ::<namespace>:: emulation_telnet_stats

DESCRIPTION

IxLoad

This command manages the statistics gathered during the test run. It can add multiple types of statistics, extract data, clear statistics from a previous run, or can add specific filters to the way the statistics are gathered.

SYNOPSIS - ::<namespace>::emulation_telnet_stats

Supported platforms Details

IxLoad

DESCRIPTION

Specifies how statistics for multiple ports are combined.Dependencies: -mode add.sum - add all of the statistics together. max - determine the maximum value.min - determine the minimum value.average - determine the average value.rate - determine the rate of change of the sum of all the statistics.maxrate - determine the maximum rate.minrate - determine the minimum rate.averagerate - determine the average rate.

Number of responses sent by the client agreeing to use the Line Mode option.telnet_total_line_mode_options_wont_sent - Number of responses sent by the client rejecting use of the Line Mode option.Sub-Options Statistics:telnet_total_suboptions_received -
