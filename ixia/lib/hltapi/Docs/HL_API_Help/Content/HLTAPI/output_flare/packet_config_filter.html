-mode

Supported platforms Details

IxNetwork, IxOS/IxNetwork-FT

DESCRIPTION

Supported with IxTclHal and IxTclNetwork (addAtmFilter choice is not valid with IxTclNetwork). When mode is create ATM specific configurations and general configurations will be set.The user can set the same pattern_atm for more pairs vpi/vci pairs (see parameters -vpi and -vci).If the user wants to set different pattern_atm for some pairs, he has to call this procedure using -mode addAtmFilter.When using -mode addAtmFilter, only ATM specific configurations will be set and all the other configurations will be ignored. These ATM specific configurations will be appended to the previous ones.This procedure must be called only once with -mode create, otherwise it will overwrite the previous configurations.
