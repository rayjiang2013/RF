l2tp_config

General

Parameter Description

is_last_subport - Functionality Extended

This parameter was used because IxAccess had a limitation to 4 subports. This limitation is no longer valid for IxNetwork.

max_outstanding

When IxNetwork is used this parameter takes values in the 1-1000 range.
When IxAccess is used this parameter takes values in the 2-2000 range.

Statistics

Parameter: For IxNetwork the statistic keys are prepended with <port_handle> because statistics can be gathered for multiple ports with only one call; for IxAccess they can be retrieved only without <port_handle> stats. The statistics are returned in both formats for backwards compatibility (with and without <port_handle>)
