Parameter Valid for Comments
IxOS IxNetwork 5.30 or earlier IxNetwork 5.40 or later
status Yes Yes Yes $::SUCCESS | $::FAILURE
log Yes Yes Yes On status of failure, gives detailed information.
handle Yes Yes Yes The name of the array to be used for reading the stats. This key is returned when more than 2000 statistics values need to be retrieved. The indices of this array are the keys that you'll find explained below, and the values are the statistics values. This method has been introduced to increase performance when returning and reading stats. Returning a huge list in the tcl console can make the console hang.
clicks Yes Yes Yes Time when the data was fetched from the hardware. The value is of local significance to the chassis.
seconds Yes Yes Yes Time when the data was fetched from the hardware. The value is the chassis' current time, in seconds since 1/1/1970 0:0:0.
waiting_for_stats Yes Yes Yes Returned only when traffic_generator is ixnetwork_540 (NextGen Traffic). If traffic statistics have arrived this key will be 0, otherwise it will be 1.
csv_file Yes Yes Yes csv file path(s) if -return_method is csv
measure_mode Yes Yes Yes detected traffic stats measure mode, the returned stats are according to this value
