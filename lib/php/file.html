Convert a testcase file from .txt to .xml

Keyword:
Status:


Procedure:
1: Click on "Choose File" Button to choose testcase file in .txt format.
2: Enter Keyword. If enter multiple Keywords separate each using ",". If do not enter anything in Keyword field, there is no keyword in default.
3: Enter Status. The default Status is 1 if enters nothing in Status field. The valid Status numbers are 1-7(1 =Draft, 2=Ready for review, 3=Review in progress, 4=Rework, 5=Obsolete, 6=Future, 7=Final)
4: Click on "Convert" button to generate "UTF-8" encoding xml format file.
5: Right click the mouse and select the "save as" to save file.
